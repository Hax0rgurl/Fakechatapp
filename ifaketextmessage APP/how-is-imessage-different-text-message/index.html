How is an iMessage different from a Text Message?

On an iPhone, an iMessage will appear with Blue Bubbles. A normal Text Message, however, will appear with Green Bubbles. There are two features that iFakeTextMessage.com provides for creating realistic messages.

First of all, when adding a new text message, select the color of the bubble by selecting either the grey, blue, or green squares besides the message.

Secondly, under the "Settings" tab, find the "Message Area" option, and select either "iMessage" or "Text Message".
